[README.md](https://github.com/user-attachments/files/29964544/README.md)# Ma Maison Review Dashboard

Static single-page dashboard prepared for cloud hosting.

## Cloudflare Pages

1. Open Cloudflare Dashboard → Workers & Pages.
2. Choose Create → Pages → Upload assets.
3. Upload this folder or the ZIP package.
4. No build command is required.

The dashboard stores imported and edited review data in the current browser's local storage. Use **Export CSV** to back up or share updates.
